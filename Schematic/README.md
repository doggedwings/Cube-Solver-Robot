## Note on A4988 Component

The A4988 symbol used in the KiCad schematic is not the exact same footprint as the physical Pololu A4988 breakout board used in the actual build. The correct breakout board footprint was not available in the KiCad library at the time. The pin functions and connections are identical, only the pin numbering differs between the schematic symbol and the physical board, so the wiring logic is fully accurate.
